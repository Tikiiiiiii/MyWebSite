## 2024.06.23
查到有滚动条的原因在于body的默认样式带有8px的边距，因此需要先清除默认样式（网上找了一个）
https://meyerweb.com/eric/tools/css/reset/
把css引入页面
```css
/* src/assets/main.css */
@import './reset.css';

body {
  width: 100vw;
  height: 100vh;
  background-image: url('/bgimg.jpg');
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
}
```
